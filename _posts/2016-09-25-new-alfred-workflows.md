[Alfred](https://www.alfredapp.com) is an award-winning app for Mac OS X which boosts your efficiency with hotkeys, keywords, text expansion and more. I use it extensively, and this week I shared a handful of new workflows that you may find interesting..

## Search Giphy

Use the keyword `gif` followed by a search term, and you will be taken to the Giphy search results in your default browser.

<a class="btn btn-primary" href="https://github.com/mathewcropper/alfred/tree/master/search-giphy" role="button">Download Search Giphy here</a>

## New Google Docs

A handful of keywords to allow you to quickly create new Google docs:

* `new doc` for a document
* `new sheet` for a new spreadsheet
* `new slide` for a new presentation
* `new form` for a new form

<a class="btn btn-primary" href="https://github.com/mathewcropper/alfred/tree/master/new-google-docs" role="button">Download New Google Docs here</a>

## Run Jekyll Server

Use `jekyll server` to quickly run the Jekyll server based on your current Finder location, and open the site in your default browser. 

You can also use `jekyll browser` if the server is already running to open the site in your browser.

<a class="btn btn-primary" href="https://github.com/mathewcropper/alfred/tree/master/run-jekyll-server" role="button">Download Run Jekyll Server here</a>
