If you find yourself using Spaces a lot on Mac OS X, you may want a quicker way to create new ones using your keyboard.

I created an [Alfred](https://www.alfredapp.com) workflow to trigger Exposé and create a new Space, using the keyword `space`.

<a class="btn btn-primary" href="https://github.com/mathewcropper/alfred/tree/master/new-space" role="button">Download the workflow here</a>

Once installed, open the Alfred prompt, type `space`, and then hit enter to trigger this workflow. You could change this to using a hotkey combo instead of using a keyword to save even more time.
